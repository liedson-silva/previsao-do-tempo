# Previsão do Tempo

## 📋 Descrição

Projeto de previsão do tempo que, ao inserir país ou cidade, exibe o clima atual, temperatura mínima e máxima, e a previsão para os próximos 5 dias.

## 🛠️ Tecnologias

- ReactJs
